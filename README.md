> **Vyne** is a premium Micro-SaaS platform that bridges the gap between urban living and sustainable agriculture. We connect neighbors to share seeds, lend specialized tools, and unlock hidden planting spaces in the concrete jungle.

---

## ✨ The Vision
Urban gardening shouldn't be a luxury. Vyne solves the "entry tax" of farming—expensive tools and lack of land—by creating a hyper-local, peer-to-peer ecosystem. Built with a **Nature-Tech** philosophy, the app combines organic UI elements with high-performance animations to make sustainability feel modern and accessible.

## 🚀 Core Features

### 🛠️ The Swap Shop
A hyper-local marketplace for tool lending and seed trading.
* **Proximity Logic:** Find a shovel or heirloom seeds within a 5-minute walk.
* **Verified Trust:** Peer-review system for "Master Lenders."

### 📍 Plot Finder
An interactive map utility for land-sharing.
* **Unlock Space:** Homeowners can list unused side-yards, rooftops, or "hell strips."
* **Micro-Leasing:** Secure agreements for neighborhood micro-farming.

### 🧬 The Seed Vault
A digital heritage log for heirloom biodiversity.
* **Lineage Tracking:** Document the history and success of shared seeds.
* **Inventory Management:** Smart alerts for seasonal planting windows.

### ⭕ Growth Circles
Localized community hubs for bulk ordering soil, coordinating harvests, and organizing "garden-mobs" to help neighbors set up new plots.

---

## 🎨 Design Philosophy: "Nature-Tech"
Vyne isn't just an app; it's an experience. We’ve utilized **GSAP (GreenSock Animation Platform)** to create a fluid, living interface:
* **Organic Motion:** Staggered entrance animations and parallax scroll effects that mimic natural growth.
* **Glassmorphic UI:** A clean, "Earthy-Premium" aesthetic using Forest Green, Sage, and Terracotta.
* **Night Garden Mode:** A sophisticated dark theme designed for low-light evening planning.

---

## 🛠️ Tech Stack

* **Frontend:** `React (Vite)` for lightning-fast HMR and performance.
* **Animations:** `GSAP` for high-end, scroll-triggered visual storytelling.
* **Styling:** `Tailwind CSS` for a modular, responsive design system.
* **Backend/Auth:** `Supabase` (PostgreSQL) for real-time data sync and secure user authentication.
* **Icons:** `Lucide React` for a clean, minimalist aesthetic.

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/vyne-app.git
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Environment Setup**
   Create a `.env` file and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key
   ```
4. **Run Development Server**
   ```bash
   npm run dev
   ```
